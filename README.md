

# INSTALLATION INSTRUCTIONS

## 1.Front End

## **Prerequisite**

 - Node js 10 +
 - npm 6 +

## Installation & setup

 - `git clone git@github.com:abhinav-TB/medbikri.git`    
 - `cd front-end`
 -   `npm install`    
 - `npm start`

## 2. Web scrapping

## **Prerequisite**

 - python 3
 
##  Installation & setup
 - clone the repository if not already done `git clone git@github.com:abhinav-TB/medbikri.git`
 - cd Scraping
 - pip install -r requirements.txt
 - python3 main.py
